# ping_exporter Helm Repository

This branch hosts the Helm chart repository for ping_exporter.

## Usage

```bash
helm repo add ping_exporter https://czerwonk.github.io/ping_exporter
helm repo update
helm install my-ping-exporter ping_exporter/ping_exporter
```

For chart source and documentation, see the [main branch](https://github.com/czerwonk/ping_exporter/tree/main/dist/charts/ping-exporter).
